## Decisión de diseño resuelta

**D1 — AuthenticationException custom vs UnauthorizedAccessException BCL**

Decisión: crear tipo custom `AuthenticationException` con propiedad `Code`. Motivo: `UnauthorizedAccessException` BCL es lanzada por .NET en escenarios de I/O y file access; capturarla en el middleware global hubiera retornado falsos HTTP 401 en errores no relacionados con autenticación.
