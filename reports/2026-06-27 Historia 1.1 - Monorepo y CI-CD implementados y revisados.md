## Aprendizajes técnicos

- **.NET SDK no disponible localmente** — todos los archivos de proyecto .NET se crearon manualmente a partir del conocimiento de plantillas. Verificación de build queda delegada a CI.
- **Gradle 8.13 (no 8.14)** — la historia especificaba 8.14 pero el cache local solo tenía 8.13. Compatible con AGP 8.10.1; se usó 8.13.
- **`JAVA_HOME` en Bash** — debe exportarse explícitamente en el mismo comando donde se usa `gradle wrapper`. No se hereda entre invocaciones del tool.
- **AndroidManifest simplificado** — sin `android:icon` / `android:roundIcon` para evitar errores de recursos faltantes (mipmap). Historia 1.3 agrega el ícono oficial.
- **`appsettings.Development.json` debe estar en git** — el `.gitignore` original lo excluía por error. Corregido: solo se ignora `appsettings.Production.json`.
- **`deferred-work.md`** — creado en `_bmad-output/implementation-artifacts/` para rastrear items diferidos entre historias.
